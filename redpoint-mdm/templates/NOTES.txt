MDM is installing. It can take a few minutes for all services to become ready.

REQUIRED - MongoDB connection secret
  MDM reads its MongoDB connection string from the secret '{{ $secret }}'
  (key: MONGO_CONNECTION_STRING). The chart references this secret but never
  creates it. If it is missing, pods stay in CreateContainerConfigError.

    kubectl create secret generic {{ $secret }} \
      --from-literal=MONGO_CONNECTION_STRING='mongodb://<user>:<password>@<host>:27017/admin?authSource=admin' \
      --namespace {{ $ns }}
